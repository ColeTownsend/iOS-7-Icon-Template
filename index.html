iOS7 Icon Template

Using the Magical Super Ellipse

This is a .sketch file that uses the iOS7 super ellipse to create the proper icon shape. It is not simply a rounded border rectangle. This was created using Grapher, and is the exact shape of the iOS 7 icon. Below the difference is illustrated by superimposing an incorrect rounded border radius over the super ellipse shape
